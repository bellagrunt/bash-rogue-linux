## Commands

1. Check path:
pwd

2. show listings:
ls
ls -a
ls -l
ls -al

3. Change Directory:
cd

4. Remove folder
sudo rm -rf filename

5. Make directory:
mkdir foldername

6. Back one level from a folder and root:
One folder:
cd ..

Root:
cd /
cd

7. Clear terminal:
clear

8. Create file:
touch filename

9. Edit file:
nano filename

10. See what's in a file:
cat filename
less filename

11. Rename file:
mv filename1 filename2

12. Copy files
cp filename1 filename2

13. Remove file:
sudo rm filename

14. Remove directory:
sudo rmdir filename
